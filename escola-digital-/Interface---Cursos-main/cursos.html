Curso 1 - Java

Neste curso, o aluno irá aprender habilidades fundamentais para programação, incluindo a sintaxe da linguagem Java, orientação a objetos, tratamento de exceções e programação concorrente. Também serão explorados tópicos avançados, como desenvolvimento web, acesso a bancos de dados e criação de interfaces gráficas

Curso 2 - JavaScript

Neste curso, o aluno irá adquirir habilidades essenciais relacionadas à programação web e ao desenvolvimento de aplicativos interativos. Isso inclui aprender a sintaxe da linguagem JavaScript, manipulação do DOM (Document Object Model) para interagir com elementos HTML, implementação de eventos e ouvintes para criar funcionalidades interativas em páginas da web.

Curso 3 - HTML e CSS

Neste curso, o aluno irá aprender a criar e estilizar páginas web de forma eficaz, autônoma e esteticamente agradável. Alguns dos assuntos que serão abordados incluem: markup HTML, formulários interativos, estilização com CSS, layout responsivo, flexbox e grid, acessibilidade web e integração de mídia.

Curso 4 - C

Neste curso, o aluno irá aprender os fundamentos essenciais da programação e desenvolvimento de software, incluindo conceitos de sintaxe, manipulação de dados, estruturas de controle, gerenciamento de memória e depuração. Também serão explorados tópicos como funções, modularidade, estruturas de dados, entrada/saída e algoritmos.
